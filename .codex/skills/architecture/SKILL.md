# Software Architecture Skill

## Core Principles

- SOLID
- Clean Architecture
- Separation of concerns
- High cohesion
- Low coupling

## Frontend

Prefer:
- feature-based architecture
- domain separation
- reusable UI primitives
- isolated business logic


## Review Checklist

Evaluate:
- maintainability
- scalability
- complexity
- developer experience
- testability
